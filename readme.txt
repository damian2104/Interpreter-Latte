Program zawiera interpereter języka Latte (odrobinę zmodyfikowanego - gramatyka znajduje się w pliku Lang.cf)

Aby uruchomić program należy wykonać komendy:
make
./Interpreter plik
(gdzie plik jest programem zapisanym w języku)

Program w obecnej wersji wypisuje na wyjście wartość zwracaną przez funkcję main bądź komunikat o błędzie jeśli jakiś wystąpił.
(Value -> NumVal Int | StrVal String | BVal Bool)

W katalogach bad/good znajdują się przykładowe programy, które kończą porażką/sukcesem.
